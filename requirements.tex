\subsection*{Recommended Requirements}

\begin{itemize}
    \item \textbf{Operating System:} Windows 10/11 (64-bit), macOS (64-bit), Linux (64-bit)
    \item \textbf{CPU:} $\geq$ Quad-core (e.g., Intel Core i5 / AMD Ryzen 5)
    \item \textbf{RAM:} $\geq$ 16GB
    \item \textbf{Storage:} $\geq$ 7GB free space (for models)
\end{itemize}


Linux users should be able to run the code on similar hardware (GPU not required), but may need to adjust installation steps due to PyTorch compatibility issues.

\section*{Python Dependencies}

It is recommended to use a virtual environment (e.g., venv, conda) to manage Python dependencies and avoid conflicts with system packages.
\begin{verbatim}
python --version  # Ensure Python 3.12.x is installed and currently active
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
\end{verbatim}

Required packages are locked and listed in \texttt{requirements.txt} and can be installed via pip:
\begin{verbatim}
(venv) pip install -r requirements.txt
\end{verbatim}

\subsection*{Linux OS Notes}

While these steps worked ok on my machines, PyTorch may not install nicely on Linux.

Please please do not mark down due to this, this is completely out of my control. The code should run without GPU support, albeit much slower.

I have specifically not used the GPU to make this as compatible as possible, please visit \url{https://pytorch.org/get-started/locally/} if the requirement cannot automatically be met.
